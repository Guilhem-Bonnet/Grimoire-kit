MODULES · DÉTAIL DES COMPOSANTS AGENTS
BMM

Builder Mind Map

Cartographie cognitive de la session. Maintient le graphe de connaissances de l'agent, les dépendances entre tâches et les hypothèses actives.

BMB

Builder Memory Bank

Banque mémoire à trois niveaux : working (volatile), episodic (session), semantic (permanent). Isolation par namespace, TTL configurable.

CIS

Contextual Intent System

Système d'interprétation des intentions. Traduit les directives de haut niveau en contrats d'exécution précis pour le SOG.

TEA

Trace Emit Agent

Agent responsable de l'émission des traces. Intercepte chaque action, l'horodate, la signe et l'envoie au Trace Ledger immutable.
