01 · MANIFESTE
01 · LA TRACE OU RIEN

Chaque action produit une preuve auditable.

Pas de magie, pas de boîte noire. Chaque décision d'agent, chaque artefact émis, chaque contrat signé est enregistré dans le ledger immutable du runtime.

02 · UN SEUL POINT D'ENTRÉE

Le SOG orchestre. Tu parles à une interface, pas à trente bots.

Le Smart Orchestrator Gateway dispatche vers des sub-agents invisibles. L'opérateur contrôle un système cohérent, pas une collection d'appels API.

03 · COMPOSABLE PAR NATURE

Agents, skills, workflows, instructions : tout est modulaire, tout est forgeable.

Archétypes, agents, workflows et skills se combinent librement, et le standard agentique gouverné active 36 patterns vérifiables. Tout est remplaçable sans toucher au runtime.

05 · FLUX

De l'intention à l'artefact.

01

Contrat

L'opérateur définit l'objectif, les outils autorisés et les critères. Le runtime valide la signature avant exécution.

02

Orchestration

Le SOG sélectionne et spawne les agents selon le contrat. Chaque agent reçoit un contexte mémoire délimité.

03

Frappe

Le runtime orchestre la séquence, gère les dépendances et enregistre chaque décision dans le trace ledger.

04

Artefact tracé

L'artefact est versioné, adressable par hash et signé. Disponible dans l'Observatory, exportable via API.
